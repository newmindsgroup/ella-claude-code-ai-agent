            What we're tracking
            
              
            We attribute tokens to tasks via timestamp windows on lifecycle events. Human-equivalent value uses industry-standard hourly rates per task type (drafts at $75/hr, research at $150/hr, strategy at $200/hr, creative at $125/hr). API cost is computed at Sonnet 4.6 list pricing — $3 per Mtok input, $15 per Mtok output, with cache reads at 10% and cache writes at 125% of input pricing. Net savings is human-equivalent cost minus api cost.

            • Approval rates — % of drafts shipped without edits.

            • Edit-distance per draft — how much you actually changed.
              • 
                
            • Sub-agent delegation graph — who called whom, how deep.
              • 
                
            • Response latency p50 / p95 — tail behavior, not averages.
              • 
                
            • Cost per outcome — per booked call, per closed deal.
